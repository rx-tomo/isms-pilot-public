# Contributing

Thank you for your interest in ISMS Pilot.

This repository is source-available for evaluation, not an open source project. Product feedback, ISMS practice feedback, questions, and collaboration ideas are welcome.

Code contributions may be reviewed case by case. Acceptance of a contribution does not grant commercial use rights beyond the [LICENSE](LICENSE).

Before opening a pull request:

1. Confirm the change does not include secrets, tokens, customer data, private logs, generated databases, screenshots containing sensitive information, or internal agent/workflow notes.
2. Run `npm run lint`.
3. Run `npm run typecheck`.
4. Run `npm run lint:messages`.
5. Run `npm run qa:public-copy`.
6. Run `npm run build`.

For commercial licensing, hosted use, redistribution, individual implementation support, or partnership discussions, contact the repository owner before relying on this code.
