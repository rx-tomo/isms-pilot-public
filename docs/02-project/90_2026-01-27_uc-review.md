### 変更が発生したUCの詳細

#### UC-05 リスクアセスメント — AI支援リスク評価（P5-2）の影響

**完了項目（変更なし）**: UC-05の全チェック項目は引き続き完了状態。

**新規追加事項**:
- P5-2（AI支援リスク評価）が SQLite リポジトリパターンで実装済み（PR #210, コミット 30bdaa8）
- `AISuggestionRepository` と `AIUsageLogRepository` が Phase 1 で @libsql/client に移行完了
- AI設定が `SQLiteConfigStore` 経由で永続化される（GAP-1 修正済み、コミット f210a4f）

**影響評価**:
- UC-05 の既存機能（リスクCRUD、マトリクス、エクスポート等）は Supabase 経由のまま動作
- AI支援機能は DATABASE_MODE=sqlite/turso 時に SQLite リポジトリで動作
- 将来の Phase 2（リポジトリ完全移行）で UC-05 のリスク CRUD も SQLite に移行予定

**UCチェックリストへの追加候補**:
```
- [ ] DATABASE_MODE=sqlite で AI支援リスク評価（提案生成・承認・拒否）が動作することを確認する
- [ ] AI使用量ログが SQLite に記録され、/settings/ai/usage で表示されることを確認する
- [ ] AI設定（プロバイダー・モデル・温度パラメータ）が永続化されることを確認する
```

#### UC-09 設定・権限管理 — DATABASE_MODE=turso 対応

**完了項目（変更なし）**: UC-09の全チェック項目は引き続き完了状態。

**新規追加事項**:
- DI Container に `turso` モードが追加され、13リポジトリが sqlite/turso 両方に対応
- DATABASE_MODE 環境変数に `turso` が有効値として追加

**影響評価**:
- UC-09 の既存設定画面は Supabase モードでは変更なし
- turso モードの設定画面は Phase 2（リポジトリ完全化）完了後に検証が必要

**UCチェックリストへの追加候補**:
```
- [ ] DATABASE_MODE=turso で DI Container が正しくリポジトリを生成することを確認する
- [ ] TURSO_DATABASE_URL 環境変数でリモート Turso DB に接続できることを確認する（Phase 4）
```
