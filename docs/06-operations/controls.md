# 管理策ライブラリ運用ガイド

本ドキュメントでは、ISO 27001 附属書A（Annex A）管理策の登録・管理・ウィザード機能について記載します。

## 1. 概要

`/settings/controls` 画面では、ISO/IEC 27001 Annex A に基づく管理策を一元管理できます。

### 機能一覧

| 機能 | 説明 |
|------|------|
| 個別登録 | 管理策を1件ずつフォームから登録 |
| 附属書Aウィザード | テンプレートから管理策を一括登録 |
| 検索・フィルタ | キーワード・カテゴリで絞り込み |
| 編集・削除 | 既存の管理策を編集または削除 |

## 2. 管理策テンプレートウィザード

### 概要

「附属書Aから追加」ボタンをクリックすると、事前登録されたテンプレートから管理策を一括投入できます。

テンプレートは `control_templates` テーブルに保存されており、多言語（ja/en）対応です。

### 適用モード

| モード | 説明 |
|--------|------|
| **insert**（新規のみ追加） | テンプレートの `template_key` に一致するレコードが組織内に存在しない場合のみ登録。既存は維持。 |
| **restore**（補完） | 削除された管理策を再作成したい場合に使用。未登録のテンプレートのみ追加。 |
| **overwrite**（上書き） | 既存の管理策（同じ `template_key`）を最新テンプレートの値で更新。タグ・説明も上書きされる。 |

### ウィザードの使い方

1. `/settings/controls` 画面を開く
2. 「附属書Aから追加」ボタンをクリック
3. キーワードでテンプレートを絞り込む（任意）
4. 登録したいテンプレートにチェックを入れる
5. 適用モードを選択
6. 「管理策を登録」ボタンをクリック
7. 結果がトースト通知で表示される

### 登録済みテンプレートの識別

- 一覧で緑色のバッジ「登録済み」が表示されるテンプレートは、同じ `template_key` を持つ管理策が既に存在
- `insert` モードでは登録済みテンプレートはスキップされる

## 3. データモデル

### iso_controls テーブル

| カラム | 説明 |
|--------|------|
| `id` | UUID 主キー |
| `organization_id` | 組織ID（RLS で分離） |
| `control_code` | 管理策コード（例: A.5.7） |
| `category` | カテゴリ名 |
| `title` | 管理策名 |
| `description` | 説明 |
| `tags` | タグ配列 |
| `template_key` | ウィザード由来のテンプレートキー（例: `annex_a.5.7.isms_lead`） |
| `created_at` | 作成日時 |
| `updated_at` | 更新日時 |

### control_templates テーブル

| カラム | 説明 |
|--------|------|
| `id` | UUID 主キー |
| `template_key` | テンプレート識別子（複合ユニーク） |
| `locale` | 言語コード（ja / en） |
| `category` | カテゴリ名 |
| `title` | 管理策名 |
| `description` | 説明 |
| `control_code` | 管理策コード（例: A.5.7） |
| `annex_reference` | 附属書参照（例: Annex A 5.7） |
| `default_tags` | デフォルトタグ配列 |
| `is_default_selected` | ウィザードで初期選択されるか |

## 4. リスク対応策との紐付け

管理策はリスク対応策（`risk_treatments`）に紐付けできます。

- `risk_control_links` テーブルで N:N のリンクを管理
- リスク対応策編集画面から関連する管理策を選択可能

## 5. 権限

### アクセス要件

以下のいずれかの条件で管理策機能を利用可能：

- `system_operator` ロール
- `org_admin` ロール
- `can_manage_controls` 権限が付与された一般ユーザー

権限がないユーザーは `/settings/controls` にアクセスできません。

## 6. トラブルシューティング

| 症状 | 対応 |
|------|------|
| ウィザードでテンプレートが表示されない | ネットワークエラーか、`control_templates` テーブルにシードデータが入っていない可能性。DBを確認してください。 |
| 「登録済み」のテンプレートが投入されない | `insert` モードでは既存テンプレートはスキップされます。`overwrite` モードを使用してください。 |
| 削除した管理策を復元したい | ウィザードで同じテンプレートを選択し、`restore` モードで再投入してください。 |
| 管理策コードが重複する | 組織内で同じ `control_code` は一意制約があります。既存を編集または削除してから追加してください。 |
