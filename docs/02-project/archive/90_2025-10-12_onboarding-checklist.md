# 2025-10-12 オンボーディング進捗チェックリスト追加メモ

## 概要
- Org Admin が初期設定からリスクアセスメント開始までの準備状況を俯瞰できるよう、ホーム画面に「オンボーディング進捗」カードを追加。
- Supabase 上の組織プロフィール、ユーザー数、部門、ISMS 適用範囲、文書、タスク、リスク件数を集計し、7 ステップの完了状況を自動判定する `OnboardingService` を実装。
- 進捗カードでは完了率・各ステップの説明・状況バッジ・次のアクションへのリンクを表示し、無料トライアル中に行うべき初期セットアップを可視化。

## 想定ユーザーフローへの影響
1. サインアップ直後に Org Admin がホームへ遷移すると、未完了のステップが「未完了」と表示され、必要な設定画面へ遷移できる。
2. 組織プロフィール／部門／適用範囲／文書／タスク／リスクを登録していくと、カード上の完了バッジがリアルタイムに更新される。
3. これにより、初回リスクアセスメント開始前に必要な準備を漏れなく実施できるチェックリストが完成し、ウォーキングスケルトンでの一連テストを実行可能になった。

## フォローアップ
- 各ステップ完了条件のしきい値を設定画面から調整できるようにする案を backlog へ追加検討。
- 招待メールやタスク自動テンプレート連携など、今後のオートメーション施策と連動させる。
